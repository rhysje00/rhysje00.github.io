Graph datasets

Collections of graphs

Working in algebraic graph theory, I often find myself using and contributing to several datasets of graphs. On this page, I list these datasets and give methods to access the data directly or indirectly.

In the table below I list the datasets of graphs I have currently been involved in creating and/or curating.


Name Description Access

Quartic Cayley graphs Complete for orders 5-1025 5-1025
Minimal Cayley graphs Complete for orders 2-511 2-511, 384
